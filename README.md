# SportBemy - Sistema de Gestión de Club Deportivo

¡Bienvenido a **SportBemy**! Esta es una solución integral desarrollada en **Java con Spring Boot** diseñada para digitalizar y optimizar la administración de un club de fútbol profesional o formativo.

##  Objetivo del Proyecto
El objetivo principal es centralizar toda la información operativa del club en una sola plataforma. El sistema permite gestionar desde el registro de usuarios y acudientes hasta el seguimiento detallado del rendimiento deportivo, salud de los jugadores y control financiero.

##  Tecnologías Principales
* **Lenguaje:** Java 17+
* **Framework:** Spring Boot 3.x
* **Base de Datos:** MySQL
* **Motor de Plantillas:** Thymeleaf (Frontend dinámico)
* **Persistencia:** Spring Data JPA / Hibernate
* **Seguridad:** Spring Security
* **Utilidades:** Lombok & Bean Validation

##  Estructura del Sistema (Módulos)
Basado en un modelo relacional robusto, el sistema se divide en:

1.  **Seguridad y Acceso:** Manejo de roles (Admin, Entrenador, Jugador, Acudiente) y autenticación segura.
2.  **Estructura Deportiva:** Gestión de categorías, equipos y perfiles técnicos de jugadores.
3.  **Entrenamiento y Rendimiento:** Control de asistencia, sedes y resultados de tests físicos periódicos.
4.  **Competición:** Programación de partidos, convocatorias y registro de estadísticas (goles, tarjetas, minutos).
5.  **Salud y Finanzas:** Seguimiento de lesiones (tratamientos y altas) y control de pagos de membresías mensuales.
